## 🎓 Aloomni

Aloomni adalah aplikasi desktop berbasis Java yang dirancang untuk membantu pengelolaan data alumni pada suatu institusi pendidikan. Proyek ini dikembangkan sebagai tugas mata kuliah dan berfokus pada perancangan antarmuka (User Interface) yang sederhana, modern, dan mudah digunakan.

## 📖 Tentang Proyek

Aloomni bertujuan untuk mempermudah proses pengelolaan data alumni melalui aplikasi desktop. Pada tahap pengembangan saat ini, proyek masih berfokus pada desain antarmuka (UI) dan struktur aplikasi sebagai dasar implementasi sistem pada tahap selanjutnya.
